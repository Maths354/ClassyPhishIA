À propos de ClassyPhishIA

Contexte

Dans un monde dans lequel les technologies sont omniprésentes dans nos vies, il est important de mettre en œuvre tous les moyens nécessaires afin de pouvoir garantir la fiabilité et la sécurité de tous les appareils que nous utilisons. C’est aujourd’hui une priorité pour les professionnels et défenseurs travaillant dans la cybersécurité de pouvoir protéger les usagers ainsi que les entreprises et les gouvernements des nombreuses menaces qu’ils les guettent.

Le phishing est une des premières principales menaces aujourd’hui en ligne d’après le rapport d’activité 2023 disponible sur cybermalveillance.gouv.fr

Notre objectif principal est de fournir un outil qui proposera plusieurs fonctionnalités d’analyse essentielles sur un site de phishing et d'ainsi de déterminer un score de confiance à partir des sites légitimes comparés.

Fonctionnement

Pour déterminer le score de fiabilité d'un site, ClassyPhishIA utilise le machine learning avec un modèle de classification par régression logistique. Le modèle est entrainé à l'aide de données collectées en ligne, qui représentent diverses informations que l'on peut retrouver sur un site

Utilisation

Après avoir récupéré l'outil sur Github, l'utilisateur doit avoir installé les différentes dépendances Python pour faire fonctionner les programmes. L'utilisateur peut ensuite lancer le serveur Flask, insérer une URL, consulter les résultats de l'analyse et signaler si besoin, le site au gouvernement.

L'usager peut également avoir la possibilité d'effectuer une analyse plus poussée d'un site, en insérant sa clé API VirusTotal via l'interface.

Avantages de ClassyPhishIA

ClassyPhishIA fournit un résultat et une analyse détaillée d'un site web à l'aide d'un score de confiance global. Plus ce score tend vers 0, moins le site est fiable. Egalement, un affichage détaillé sur les informations collectées est consultable dans la page des résultats. Pour finir, un tableau référençant diverses informations sur le site analysé est visionnable, permmettant ainsi de renforcer l'avis de l'utilisateur sur le site en question.
